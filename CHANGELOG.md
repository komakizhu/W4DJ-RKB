# W4DJ RKB 更新日志

## v2.1.0

### Added

- 新增转换前汇总确认窗口，一次查看新增、已存在、将跳过和错误文件数量。
- 新增预计输出大小，开始转换前即可了解大致磁盘空间需求。
- 新增转换历史记录，保存任务时间、文件数量、失败文件和输出目录。
- 新增失败任务重试，可只重试失败歌曲，并支持导出错误报告。

### Improved

- 确认窗口与实际转换使用同一份预检结果，避免确认后统计与实际执行不一致。
- 历史记录按任务集中展示，方便追踪多次转换结果。
- 输出模式和 WAV/AIFF 选择操作更稳定，快速操作时界面响应更连贯。
- 更新使用说明，补充预检确认、历史记录和失败重试的使用方式。

### Fixed

- 修复反复点击兼容模式时滑动动画不断重新播放的问题。
- 修复快速切换 WAV 和 AIFF 时按钮短暂卡顿或状态不同步的问题。
- 修复部分任务失败后无法明确查看失败文件的问题。

### Breaking Changes

- 无。

## v2.0.1

### Added

- 支持两个同步任务同时配置和执行，适合批量处理网易云音乐与 SoundCloud 曲库。
- 支持兼容模式输出 MP3，以及无损模式输出 WAV 或 AIFF。
- 支持网易云音乐 `.ncm` 解密转换，并支持常见音频文件处理。
- 支持保留歌曲元数据和封面，生成适合 DJ 软件和设备使用的音频文件。
- 提供中英文界面、浅色/深色主题和拖放导入等基础操作体验。
- 提供 Apple Silicon、Intel Mac 和 Windows 安装包。

### Improved

- 优化批量转换进度、任务状态和输出目录配置体验。
- macOS 安装包使用易懂的 Apple Silicon 与 Intel 文件名，便于选择正确版本。

### Fixed

- 修复多来源音频转换和不同输出格式处理中的稳定性问题。
- 修复部分元数据、封面和输出文件命名在批量处理时不一致的问题。

### Breaking Changes

- 无。
