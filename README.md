# W4DJ RKB v1.0.0 - 如果我是DJ 🎧

基于Slipstream-Max-w4dj的首个原生桌面客户端正式版发布！W4DJ 是一款专为 Rekordbox 曲库同步与无损解密打造的桌面视窗工具，原生兼容 MacOs 与 Windows 平台。

![w4dj](imgs/w4dj.png)

### 🚀 核心功能与亮点

- **🎵 网易云音乐一键同步**：手动指定的本地下载文件夹，一键自动同步至自定义文件夹中。
- **🔄 无损音频输出控制**：
  - **兼容模式**：输出高品质 320kbps MP3 音频，保留元数据。
  - **无损模式**：支持 24-bit / 48000Hz 采样率 of WAV 或 AIFF 无损格式输出，完美兼容 CDJ-350 / XDJ-700 及以上硬件。
- **🏷️ 元数据自动重写**：自动将原曲解密，并将歌手、歌曲名称、专辑封面等 ID3 / FLAC 数据完整写回生成的标准音频中。
- **🎛️ 简约交互设计**：精美的沉浸式毛玻璃窗体设计，支持底部日志折叠收纳、自动记忆上次的选择路径与格式。
- **⚡ 多线程并发**：后端采用 Rust `rayon` 多线程并发引擎，数千首音乐同步仅需眨眼之间。

### 💾 安装包下载

- **macOS (Intel/M系列通用)**: 下载下方 `W4DJ RKB_1.0.0_x64.dmg` 双击安装拖拽入文件夹。
- **Windows (x64)**: 下载下方 `W4DJ RKB_1.0.0_x64-setup.exe` 双击安装。

### ⚠️ 首次运行说明（系统安全提示解决办法）
#### 🍏 macOS 提示“已损坏，打不开”解决办法
这是 macOS Gatekeeper 门禁保护 of 正常安全限制，可通过以下任意方式解锁：
* **【推荐】免终端方法**：双击打不开报错后，打开 Mac 的 **「系统设置」 -> 「隐私与安全性」**，往下拉找到 *“已阻止打开 W4DJ RKB，因为它不是来自可信的开发者”* 提示，直接点击右侧的 **「仍要打开」** 按钮，输入电脑开机密码即可。
* **【备选】一键命令行解锁**：在终端运行下方这行指令即可秒开：
  ```bash
  xattr -cr /Applications/W4DJ\ RKB.app
  ```

#### 🔌 Windows 提示“Windows 已保护你的电脑”解决办法
双击运行 `.exe` 安装程序时若弹出蓝色防护盾弹窗：
* 直接点击弹窗上的 **「更多信息」 (More info)** 文字链。
* 随后点击右下角出现的 **「仍要运行」 (Run anyway)** 按钮，即可自动进行安装。
