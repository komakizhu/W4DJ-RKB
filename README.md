# W4DJ v0.1.0 - 如果我是DJ 🎧

首个原生桌面客户端正式版发布！W4DJ 是一款专为 DJ 硬件曲库同步与无损解密打造的桌面视窗工具，采用 macOS 原生 Liquid-Glass (流体毛玻璃) 视觉美学设计，并原生兼容 Windows 平台。

### 🚀 核心功能与亮点

- **🎵 网易云音乐一键同步**：手动指定的本地下载文件夹，一键自动同步至自定义文件夹中。
- **🔄 无损音频输出控制**：
  - **兼容模式**：输出高品质 320kbps MP3 音频，保留基本元数据。
  - **无损模式**：支持 24-bit / 48000Hz 采样率 of WAV 或 AIFF 无损格式输出，完美兼容 CDJ-350 / XDJ-700 及以上硬件。
- **🏷️ 元数据自动重写**：自动将原曲解密，并将歌手、歌曲名称、专辑封面等 ID3 / FLAC 数据完整写回生成的标准音频中。
- **🎛️ 极致交互设计**：精美的沉浸式毛玻璃窗体设计，支持底部日志折叠收纳、自动记忆上次的选择路径与格式。
- **⚡ 多线程并发**：后端采用 Rust `rayon` 多线程并发引擎，数千首音乐同步仅需眨眼之间。

### 💾 安装包下载

- **macOS (Intel/M系列通用)**: 下载下方 `W4DJ_0.1.0_x64.dmg` 双击安装拖拽入文件夹。
- **Windows (x64)**: 下载下方 `W4DJ_0.1.0_x64-setup.exe` 双击安装。

*注：Mac 端的兼容模式（MP3 输出）依赖本地已安装 `ffmpeg` (可以通过 brew install ffmpeg 安装)。*
